国际化书写

所有年龄段、文化和教育水平的用户都可能使用产品的英文版。简单明了的英文能让任何人都可以理解你的产品。

在 RTL 的语言中，谨慎的使用项目的「左」和「右」

在本地化时，从右到左的语言可能需要镜像化

指定性别

有时候用户的性别是未知的。在英文中，就可以使用不区分性别的文字「they」。不同的语言则需要却别对待。

当涉及到性别时：

  • 不要合并性别，比如使用「他／她」。
  • 如果必须指定性别，比如引用某人的姓名时，可以使用 Components for Unicode (ICU) 的 API 或等效的软件库来提供准确的翻译说明。
