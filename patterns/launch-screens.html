启动页类型

当应用在加载时，如果展示出空白页面会增加用户对加载时长的感觉。因此考虑使用占位 UI 或品牌启动页来替代。

占位 UI是最无缝的启动方式，可以同时用于应用启动和应用内页面跳转。

品牌启动页可以提供短暂的品牌展示，专注于内容展示而不受 UI 的约束。

占位 UI

当应用的加载时间较短或者不需要很强的品牌识别时，占位 UI 最小化了 UI 过度，增强了应用的响应速度。

启动时仅展示核心元素（例如状态栏、应用栏和底部列表），在加载完成前不展示内容。

品牌启动页

品牌启动页可以在应用启动时展示品牌形象，在加载时展示品牌 LOGO 或其他品牌识别元素来获得最佳效果。避免在页面上展示出了品牌 LOGO 之外的其他文字，合适的时候也可以展示宣传标语。

品牌启动页

用于初始化应用（从桌面冷启动）时的启动页，避免在应用运行时或从其他应用切换回来时展示。
